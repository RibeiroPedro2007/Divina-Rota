# Divina Rota

Projeto desenvolvido em 2025 para um grupo da faculdade canção nova, com o objetivo de **auxiliar pessoas a encontrarem caronas** e se deslocarem para pontos turísticos da região.

## Sobre o projeto:
O **Divina Rota** é um site voltado para turismo e mobilidade local, permitindo que usuários planejem viagens e compartilhem caronas de forma prática e rápida.  
Não utiliza banco de dados, armazenando temporariamente as informações no navegador via **JavaScript** e **cookies**.

## Observação:
Este projeto foi desenvolvido como adaptação de uma base anterior (TCC Teen Finance), mas com funcionalidades e objetivos totalmente diferentes.

## Tecnologias utilizadas:
- **HTML**  
- **CSS**  
- **JavaScript**  

## Considerações finais:
O projeto demonstra **capacidade de adaptação e evolução** de código, mostrando como transformar uma base existente em uma solução nova para outro propósito.
